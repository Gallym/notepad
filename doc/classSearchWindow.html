Detailed Description

Derived class from MyWindow for searching notes This class implements and extends functionality of parent MyWindow. SearchWindow class introduces Window with Search functionaly. User can search within imported notes by Name, Keyword, Category or Tag. User can see results using paging functionality. User can export searching result to file which will have few (more than one) notes stored in one file. Exported file can be parse again and notes will be imported into program. More about import.
